Pattern Matching with Wildcards

Implement wildcard pattern matching with support for '?' and '*'.

- '?' matches any single character
- '*' matches any sequence of characters (including empty sequence)

The matching should cover the entire input string (not partial).

## Input Format
- First line: string s (the text to match)
- Second line: string p (the pattern with wildcards)

## Output Format
- "true" if pattern matches the entire string
- "false" otherwise

## Constraints
- 0 ≤ s.length ≤ 2000
- 0 ≤ p.length ≤ 2000
- s contains only lowercase English letters
- p contains lowercase English letters, '?' or '*'

## Sample Input 1
```
aa
a
```
## Sample Output 1
```
false
```

## Sample Input 2
```
aa
*
```
## Sample Output 2
```
true
```

## Sample Input 3
```
cb
?a
```
## Sample Output 3
```
false
```

## Sample Input 4
```
adceb
*a*b
```
## Sample Output 4
```
true
```

## Sample Input 5
```
acdcb
a*c?b
```
## Sample Output 5
```
false
```
