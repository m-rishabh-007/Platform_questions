Sum of Array of 5 Integers

## Problem Statement
Write a function `sumOfFive` that takes an array of 5 integers and returns their sum.

## Input Format
- Five space-separated integers on a single line.

## Output Format
- A single integer: the sum of the five input numbers.

## Constraints
- Each number is an integer.
- Numbers can be negative, zero, or positive.

## Sample Input 1
```
1 2 3 4 5
```
## Sample Output 1
```
15
```
## Sample Input 2
```
-1 0 1 2 -2
```
## Sample Output 2
```
0
```
